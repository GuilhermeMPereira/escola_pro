Software HDR

Somente software disponível gratuitamente e totalmente funcional é listado. Isso pode incluir versões gratuitas de software comercial, desde que os resultados sejam não prejudicado por marcas d'água ou outras limitações, e o software não expira após o período experimental. A funcionalidade da versão gratuita pode ser diferente de a versão comercial completa, mas ainda deve ser suportada e estar em desenvolvimento.

Capacidades:
Criar, Editar, Ver, Mapa de tons.
Sistema operacional suportado:
LINUX, Mac OS X, MS Windows.
Preço/Licença:
Livre (como na fala), Grátis (como cerveja).

hdrgen

hdrgen é o mecanismo de composição HDR por trás do Photosphere e do WebHDR. Binários pré-compilados estão disponíveis para diversas plataformas. Como é o irmão maior Photosphere, as imagens são alinhadas automaticamente compensar o movimento da câmera.

Formatos Capacidades Sistemas Licença
RGBE, LogLuv TIFF, OpenEXR

suporte

suporte é um gerenciador de fotos HDR multiplataforma. Ele também se interessa por uma faixa de baixa dinâmica Gerenciador de fotos (LDR) com suporte aos formatos de imagem JPEG e TIFF padrão. bracket pode ser usado para criar, visualizar, analisar, calibrar, mapa de tons e exibir imagens HDR. Permite a conversão entre diferentes formatos de imagem, e suporta tarefas básicas de edição de imagens, como rotação, corte, inversão e redimensionamento.

Formatos Capacidades Sistemas Licença
RGBE, LogLuv TIFF, FP TIFF, OpenEXR

Imagem

Imagem nasceu na comunidade fotográfica alemã. Ele oferece uma interface polida para gerar e manipular imagens HDR.

Formatos Capacidades Sistemas Licença
RGBE, LogLuv TIFF, FP TIFF, OpenEXR, PFM

Fotosfera

Fotosfera (ex Photophile) é um navegador de imagens e software de catalogação que permite a criação de HDR imagens de imagens entre colchetes de exposição portátil.

Formatos Capacidades Sistemas Licença
RGBE, LogLuv TIFF, TIFF de 48 bits, FP TIFF, OpenEXR

Ferramentas PF

PFSTools is a set of command line (and one GUI) programs for reading, writing, manipulating and viewing high-dynamic range (HDR) images and video frames. All programs in the package exchange data using a simple generic file format (pfs) for HDR data. The concept of the pfstools is similar to netpbm package for low-dynamic range images. pfstools are supplemented by pfscalibration and pfstmo for creating HDR images, and for tone-mapping operations, resp.

Formats Capabilities Systems License
RGBE, FP TIFF, OpenEXR, PFM, LogLux TIFF, PFS

Luminance HDR

Luminance HDR, formerly known as Qtpfsgui, is an open source graphical user interface application that aims to provide a workflow for HDR imaging.

Formats Capabilities Systems License
RGBE, FP TIFF, OpenEXR, LogLux TIFF, PFS

CinePaint

CinePaint, anteriormente conhecido como Film Gimp é uma bifurcação do popular editor de imagens GPL The Gimp. Os principais grupos-alvo são a indústria cinematográfica e também fotógrafos profissionais que exigem uma profundidade de cor maior do que a disponível com editores de imagem padrão.

Formatos Capacidades Sistemas Licença
RGBE, FP TIFF, OpenEXR e muitos mais...

Radiância

Radiância é um sistema de software de rastreamento de raios altamente preciso. Produz imagens HDR sintéticas e é capaz de manipular HDR imagens.

Formatos Capacidades Sistemas Licença
RGBE, XYZE, TIFF

Ferramentas FDR

Ferramentas FDR permite combinar imagens entre colchetes de exposição em imagens HDR e LDR mapeadas por tom de uma forma que permite que a superexposição e o ruído desapareçam.

Formatos Capacidades Sistemas Licença
RGBE, OpenEXR, TIFF
